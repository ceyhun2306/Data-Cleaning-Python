# Data-Cleaning-Python

A collection of data cleaning and preprocessing techniques implemented using Python, focusing on improving data quality and preparing datasets for accurate analysis.

---

## Project Overview
This repository contains practical implementations of essential data cleaning techniques using Python. The goal is to transform raw, unstructured, and imperfect data into clean, reliable, and analysis-ready datasets.

The projects demonstrate how to handle common real-world data issues such as missing values, duplicates, outliers, and skewed distributions.

---

## Topics Covered

### 1. Handling Missing Data
- Identifying missing values in datasets  
- Applying different strategies:
  - Removal of missing data  
  - Filling with statistical measures (mean, median, mode)  
- Understanding the impact of missing data on analysis  

---

### 2. Handling Duplicates
- Detecting duplicate records  
- Removing redundant data  
- Ensuring dataset consistency and integrity  

---

### 3. Handling Outliers
- Identifying outliers using statistical methods  
- Applying techniques such as IQR and visualization  
- Understanding how outliers affect analysis results  

---

### 4. Handling Skewness
- Detecting skewed distributions  
- Applying transformations (log, sqrt, etc.)  
- Improving data distribution for better modeling and analysis
